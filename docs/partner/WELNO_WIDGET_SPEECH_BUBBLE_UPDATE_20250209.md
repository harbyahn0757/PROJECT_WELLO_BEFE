# Welno RAG Chat 위젯 – 말풍선(채팅 버블) UI 수정 안내

**대상:** Welno RAG Chat 위젯 연동 파트너 (메디링스 등)  
**작성일:** 2025-02-09  
**목적:** 우측 하단 말풍선 너비·가독성 개선 사항 안내

---

## 1. 수정 요약

- **현상:** 말풍선이 가로 너비를 거의 쓰지 못하고, 글자가 2~3자씩만 줄바꿈되어 세로로 과도하게 길게 보이던 문제가 있었습니다.
- **원인:** 말풍선이 채팅 버튼과 같은 좁은 컨테이너(약 56px) 안에 있어, `max-width: 80vw`가 실제로는 56px로만 적용되고 있었습니다.
- **조치:** 말풍선을 **뷰포트 기준 고정 위치**로 배치하고, **최소·최대 너비**와 **줄바꿈 방식**을 조정했습니다.

---

## 2. 변경 내용 (Welno 측 적용 완료)

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 위치 기준 | 버튼 컨테이너 기준 (absolute) | 화면(뷰포트) 기준 (fixed) |
| 최소 너비 | 없음 | 240px (모바일 200px) |
| 최대 너비 | 80vw | min(80vw, 320px) |
| 줄바꿈 | word-break: keep-all | word-break: normal |

이제 말풍선은 **최소 240px 너비**로 표시되며, 문장이 자연스럽게 줄바꿈됩니다.

---

## 3. 파트너 측 조치

- **별도 조치 없음.**  
  위젯 스크립트를 기존과 동일한 URL로 로드하시면 됩니다.
  - 예: `https://welno.kindhabit.com/welno-api/static/welno-rag-chat-widget.min.js`
- 스크립트를 새로 배포했으므로, **브라우저 캐시**에 따라 이전 버전이 보일 수 있습니다.  
  문제가 있으면 **강력 새로고침(Ctrl+Shift+R / Cmd+Shift+R)** 또는 **캐시 비우기** 후 확인해 주세요.
