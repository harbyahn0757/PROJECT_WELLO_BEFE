현재 변경사항이 스테이징되지 않은 상태입니다:

**권장 커밋 메시지**:
```bash
git add docs/
git add -u  # 삭제된 파일 스테이징
git commit -m "docs: 프로젝트 문서 및 스크립트 재구조화

- 작업내역 폴더를 docs로 통합 (날짜별 분류)

- 전체 문서 인덱스 및 스크립트 가이드 생성
- 루트 디렉토리 임시 파일 정리
"
```

### 레거시 참조
일부 문서나 스크립트가 `작업내역/` 경로를 참조할 수 있습니다:
- 영향 범위: 최소 (대부분 독립 실행형)
- 필요시 경로 업데이트: `작업내역/` → `docs/YYYY-MM-DD_작업명/`

### 스크립트 경로 변경
이동된 스크립트를 사용하는 경우 경로 업데이트 필요:
```python
# Before
from 작업내역/...테스트스크립트/test_rag_real import ...

# After  
from docs.scripts.test_scripts.test_rag_real import ...
```
