## ⚠️ 접속이 안 될 때 확인 사항

### 1. 프론트엔드 서버 실행 확인
```bash
# 포트 9282 사용 중인지 확인
lsof -ti:9282

# 프로세스 확인
ps aux | grep "react-scripts\|craco" | grep -v grep
```

### 2. 백엔드 서버 실행 확인
```bash
# PM2 상태 확인
pm2 status

# 백엔드 API 테스트
curl http://localhost:8082/api/v1/patients/3a96200c-c61a-47b1-8539-27b73ef2f483?hospital=KHW001
```

### 3. 브라우저 캐시 문제
- 브라우저에서 `Ctrl + Shift + R` (Windows/Linux) 또는 `Cmd + Shift + R` (Mac)으로 강력 새로고침
- 개발자 도구(F12) 열고 Network 탭에서 "Disable cache" 체크

### 4. 서버 재시작
```bash
# 프론트엔드 서버 종료 후 재시작
cd /home/workspace/PROJECT_WELLO_BEFE/planning-platform/frontend
npm run kill-port
npm start
```

### 5. 포트 충돌 확인
```bash
# 다른 프로세스가 9282 포트를 사용하는지 확인
lsof -i:9282
```

## 🔧 문제 해결

### 서버가 실행되지 않을 때
```bash
cd /home/workspace/PROJECT_WELLO_BEFE/planning-platform/frontend
npm install  # 의존성 재설치
npm start
```

### 빌드 에러가 발생할 때
```bash
# node_modules 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install
npm start
```

### 백엔드 연결 문제
```bash
# 백엔드 서버 재시작
pm2 restart WELLO_BE

# 백엔드 로그 확인
pm2 logs WELLO_BE --lines 50
```
