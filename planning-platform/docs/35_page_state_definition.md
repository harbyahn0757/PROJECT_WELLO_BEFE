# 페이지 상태 정의

## 상태 분류

### 1. 질문 상태 (Question State)
데이터가 없거나 수집이 필요한 상태
- 데이터 없음
- 데이터 수집 필요
- 인증 필요
- 초기 로딩 중

### 2. 트렌드 상태 (Trend State)
건강 추이 데이터를 보여주는 상태
- 건강지표 추이 분석
- 의료기록 타임라인
- AI 분석

## 레이아웃 적용

### 질문 상태
- `ContentLayoutWithoutHeader` 또는 간단한 레이아웃
- 헤더 없이 질문/안내 메시지 표시

### 트렌드 상태
- `ContentLayoutWithHeader` 사용
- 헤더 + 토글 + 컨텐츠 구조
