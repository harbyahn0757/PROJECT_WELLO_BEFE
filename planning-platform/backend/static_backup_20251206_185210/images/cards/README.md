# 카드 이미지 저장 위치

이 폴더에는 메인 페이지 카드의 오른쪽에 표시되는 일러스트 이미지를 저장합니다.

## 이미지 파일명 규칙

- `trends-chart.png` - 검진 결과 추이 카드용 (오렌지 그래프 일러스트)
- `health-habit.png` - 건강습관 만들기 카드용 (요가 일러스트)
- `checkup-design.png` - 검진항목 설계하기 카드용 (의사 일러스트)

## 이미지 사양

- 크기: 80x80px (카드 내부 표시 크기)
- 형식: PNG (투명 배경 권장)
- 비율: 1:1 (정사각형)

## 이미지가 없을 때

이미지가 없거나 로드 실패 시 회색 placeholder가 자동으로 표시됩니다.
