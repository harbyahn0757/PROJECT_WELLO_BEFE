# 소라임 폰트 파일 저장 위치

## 폰트 파일 저장 방법

이 폴더에 다음 파일들을 저장하세요:

- `Soraime-Regular.woff2` (또는 `.woff`, `.ttf`)
- `Soraime-Bold.woff2` (또는 `.woff`, `.ttf`)

## 지원 형식

- `.woff2` (권장)
- `.woff`
- `.ttf`

## 폰트 파일이 없을 경우

폰트 파일이 없으면 `Noto Sans KR`이 대체 폰트로 사용됩니다.

## 적용 위치

폰트는 다음 위치에서 사용됩니다:
- 아이템 설명 텍스트 (카드 설명 부분)
